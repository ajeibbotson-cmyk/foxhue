# Foxhue — Client Asset Brief

**Project**: Foxhue Website
**Prepared by**: Foxhue Ltd
**Date**: February 2026
**Contact**: andrew@foxhue.com

---

## Overview

This document outlines the assets needed to build the Foxhue website. Since this is a self-referential project (Foxhue building its own site), this checklist serves as an internal tracking tool to ensure all required content is gathered before final production.

---

## Brand Assets

- [ ] **Logo — SVG format** (primary lockup, symbol only, and wordmark)
- [ ] **Logo — PNG format** (transparent background, minimum 2x resolution)
- [ ] **Favicon** (ICO, 32x32 and 16x16, plus 180x180 apple-touch-icon)
- [ ] **Open Graph image** (1200x630px for social media sharing)
- [ ] **Brand colour palette confirmation** (primary: #1A1A2E, accent: #E94560, background: #F8F8FA)
- [ ] **Typography files** (DM Serif Display and Inter — confirm Google Fonts is acceptable or provide self-hosted files)
- [ ] **Brand guidelines document** (if existing, beyond what is in project.json)

---

## Photography

- [ ] **Team headshots** (minimum 3 — consistent style, background, and lighting)
- [ ] **Studio/workspace photos** (2-4 images showing the working environment)
- [ ] **Candid working shots** (team collaboration, creative process moments)
- [ ] **Equipment/tools** (optional — design tools, screens showing work in progress)

**Specifications**:
- Minimum resolution: 2000px on longest edge
- Format: JPEG or PNG (no heavy compression)
- Consistent lighting and colour grading preferred
- Landscape orientation preferred for hero images

---

## Portfolio / Case Study Assets

- [ ] **Project screenshots** (minimum 4-6 completed projects)
- [ ] **Client logos** (SVG preferred, for portfolio grid / client list)
- [ ] **Before/after comparisons** (if applicable to any redesign projects)
- [ ] **Results/metrics** (traffic increases, conversion improvements, ROI data)
- [ ] **Client testimonials** (written approval for each, with attribution name and company)
